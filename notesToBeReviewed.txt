To store the values from input fields in an array when a form is submitted and save it to local storage, you can follow these steps:

Add an event listener to the form that listens for the submit event.
javascript
Copy code
const form = document.querySelector('form');
form.addEventListener('submit', handleSubmit);
In the submit event handler function, create an array to store the values from the input fields. You can use the FormData API to get the values of all the input fields in the form.
javascript
Copy code
function handleSubmit(event) {
  event.preventDefault();
  const formData = new FormData(event.target);
  const formValues = {};
  for (let [name, value] of formData.entries()) {
    formValues[name] = value;
  }
  // Store the formValues array in local storage
  localStorage.setItem('formValues', JSON.stringify(formValues));
}
Save the array to local storage using the setItem method. Convert the array to a string using the JSON.stringify method, since local storage can only store strings.

To loop through the array and add the contents of properties to elements in the DOM, you can retrieve the array from local storage using the getItem method and parse it back into an array using the JSON.parse method.

javascript
Copy code
const formValues = JSON.parse(localStorage.getItem('formValues'));
for (let key in formValues) {
  const element = document.querySelector(`[name="${key}"]`);
  element.value = formValues[key];
}
Use a for...in loop to loop through the properties of the array and get the element with a matching name attribute using the querySelector method.

Set the value property of the element to the corresponding value in the array.

Here's the complete code:

javascript
Copy code
const form = document.querySelector('form');
form.addEventListener('submit', handleSubmit);

function handleSubmit(event) {
  event.preventDefault();
  const formData = new FormData(event.target);
  const formValues = {};
  for (let [name, value] of formData.entries()) {
    formValues[name] = value;
  }
  localStorage.setItem('formValues', JSON.stringify(formValues));
}

const formValues = JSON.parse(localStorage.getItem('formValues'));
for (let key in formValues) {
  const element = document.querySelector(`[name="${key}"]`);
  element.value = formValues[key];
}
Note: This code assumes that all input fields in the form have a name attribute.
